# Flexnative

UI blocks and patterns for product interfaces. Copy-paste sections and components built on [shadcn/ui](https://ui.shadcn.com) and Tailwind CSS. Designed to be extended and owned.

Blocks are page sections. Patterns are smaller UI compositions for everyday interfaces. Browse both on the site, preview with real props, and copy the source into your projects. **This repo is the open source project behind [ui.flexnative.com](https://ui.flexnative.com).**
